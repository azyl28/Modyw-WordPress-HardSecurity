== Description ==
Profesjonalny motyw WordPress dla firmy IT i cyberbezpieczeństwa. 
Nowoczesny dark theme z akcentami #66CCFF i #FF6B9D.
PEŁNA INTEGRACJA Z ELEMENTOREM!

== Funkcje v2.1 ==
- Przycisk IMPORTUJ STRONY w panelu admina
- Sprawdza czy Elementor jest zainstalowany
- Wtyczka importu treścidemo wbudowana w motyw
- Tworzy stronę główną + Blog + Menu + 6 artykułów
- Pełne wsparcie dla Elementor
- Włącz/wyłącz każdą sekcję osobno
- Animacje przy przewijaniu (AOS)
- Sekcje: Hero, Usługi, Artykuły, Partnerzy, Stats, Opinie, O nas, Kontakt

== Installation ==
1. Zainstaluj i aktywuj motyw
2. Przejdź do Wygląd > Importuj Strony

== IMPORT STRONY (nowe!) ==
1. Zainstaluj Elementor (wymagany!)
2. Idź do Wygląd > Importuj Strony
3. Kliknij "Importuj Strony i Treści"
4. Gotowe! Strona główna, Blog, Menu, 6 artykułów

BEZ ELEMENTORA IMPORT NIE DZIAŁA!

== ELEMENTOR - jak używać ==
1. Po imporcie idź do Strony > Strona główna
2. Kliknij "Edytuj w Elementorze"
3. Edytuj dowolnie!
